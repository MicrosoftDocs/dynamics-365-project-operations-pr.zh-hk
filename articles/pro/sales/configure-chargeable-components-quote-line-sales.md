# <a name="configure-the-chargeable-components-of-a-quote-line---lite"></a><span data-ttu-id="fae72-103">設定報價明細的應收費元件 - 精簡</span><span class="sxs-lookup"><span data-stu-id="fae72-103">Configure the chargeable components of a quote line - lite</span></span>

<span data-ttu-id="fae72-104">_**適用於：** 精簡部署 - 交易至開立預估發票_</span><span class="sxs-lookup"><span data-stu-id="fae72-104">_**Applies To:** Lite deployment - deal to proforma invoicing_</span></span>

<span data-ttu-id="fae72-105">專案型報價明細有 *內含* 元件和 *應收費* 元件的概念。</span><span class="sxs-lookup"><span data-stu-id="fae72-105">A project-based quote line has the concept of *included* components and *chargeable* components.</span></span>

<span data-ttu-id="fae72-106">內含元件受限於下列各項：</span><span class="sxs-lookup"><span data-stu-id="fae72-106">Included components are subject to:</span></span>

  - <span data-ttu-id="fae72-107">帳務方式和客戶分割</span><span class="sxs-lookup"><span data-stu-id="fae72-107">Billing method and customer splits</span></span>
  - <span data-ttu-id="fae72-108">不得超過限制</span><span class="sxs-lookup"><span data-stu-id="fae72-108">Not-to-exceed limits</span></span> 
  - <span data-ttu-id="fae72-109">專案型報價明細上定義的發票週期設定</span><span class="sxs-lookup"><span data-stu-id="fae72-109">Invoice frequency settings defined on the project-based quote line</span></span>

<span data-ttu-id="fae72-110">您可以使用 **帳單類型** 欄位將一部分內含元件標示為應收費。</span><span class="sxs-lookup"><span data-stu-id="fae72-110">A subset of the included components can be marked as chargeable using the **Billing Type** field.</span></span> <span data-ttu-id="fae72-111">**帳單類型** 欄位是選項組，允許在報價明細的情境下使用下列值：</span><span class="sxs-lookup"><span data-stu-id="fae72-111">The **Billing Type** field is an option-set that allows the following values in the context of a quote line:</span></span>

  - <span data-ttu-id="fae72-112">應收費</span><span class="sxs-lookup"><span data-stu-id="fae72-112">Chargeable</span></span>
  - <span data-ttu-id="fae72-113">不應收費</span><span class="sxs-lookup"><span data-stu-id="fae72-113">Non-chargeable</span></span>

<span data-ttu-id="fae72-114">您可以在工作、角色和交易類別上定義應收費元件。</span><span class="sxs-lookup"><span data-stu-id="fae72-114">Chargeable components can be defined on tasks, roles, and transaction categories.</span></span>

<span data-ttu-id="fae72-115">可收費率是在報價明細的工作上所定義，並套用至該明細所包含的所有交易分類。</span><span class="sxs-lookup"><span data-stu-id="fae72-115">Chargeability is defined on the tasks for a quote line and applies to all transaction classes included on that line.</span></span> <span data-ttu-id="fae72-116">如果 **包含工作** 欄位已設定為 **整個專案** 或保持空白，則沒有提供 **應收費工作** 索引標籤。</span><span class="sxs-lookup"><span data-stu-id="fae72-116">If the **Include Tasks** field is set to **Entire project** or left blank, the **Chargeable Tasks** tab isn't available.</span></span>

<span data-ttu-id="fae72-117">報價明細的角色上所定義的可收費率只能套用至 **時間** 交易分類。</span><span class="sxs-lookup"><span data-stu-id="fae72-117">Chargeability is defined on roles for a quote line and only applies to the **Time** transaction class.</span></span> <span data-ttu-id="fae72-118">如果 **包含時間** 欄位在專案報價明細上設定為 **否**，則沒有提供 **應收費角色** 索引標籤。</span><span class="sxs-lookup"><span data-stu-id="fae72-118">If the field, **Include Time** is set to **No** on a project quote line, the **Chargeable Roles** tab isn't available.</span></span>

<span data-ttu-id="fae72-119">可收費率是在報價明細的交易類別上所定義，並且只能套用至 **費用** 交易分類。</span><span class="sxs-lookup"><span data-stu-id="fae72-119">Chargeability is defined on transaction categories for a  quote line and only applies to the **Expense** transaction class.</span></span> <span data-ttu-id="fae72-120">如果 **包含費用** 欄位在專案報價明細上設定為 **否**，則沒有提供 **應收費類別** 索引標籤。</span><span class="sxs-lookup"><span data-stu-id="fae72-120">If the field, **Include Expenses** is set to **No** on a project quote line, the **Chargeable Categories** tab isn't available.</span></span>

### <a name="update-a-project-task-to-be-chargeable-or-non-chargeable"></a><span data-ttu-id="fae72-121">將專案工作更新為應收費或不應收費</span><span class="sxs-lookup"><span data-stu-id="fae72-121">Update a project task to be chargeable or non-chargeable</span></span>

<span data-ttu-id="fae72-122">專案工作在能使下列設定變成可行的特定專案型報價明細內容中可以是應收費，也可以是不應收費：</span><span class="sxs-lookup"><span data-stu-id="fae72-122">A project task can be chargeable or non-chargeable in the context of a specific project-based quote line, which makes the following setup possible:</span></span>

<span data-ttu-id="fae72-123">如果專案型報價明細包含 **時間** 和工作 **T1**，則工作會與報價明細建立應收費關聯。</span><span class="sxs-lookup"><span data-stu-id="fae72-123">If a project-based quote line includes **Time** and the task **T1**, the task is associated to the quote line as chargeable.</span></span> <span data-ttu-id="fae72-124">如果有第二個報價明細，其中包含 **費用** 時，您可以將報價明細的 **T1** 工作關聯為不應收費。</span><span class="sxs-lookup"><span data-stu-id="fae72-124">If there is a second quote line that includes **Expenses**, you can associate the **T1** task on the quote line as non-chargeable.</span></span> <span data-ttu-id="fae72-125">結果是，工作中所記錄的所有時間都是應收費的，而工作中所記錄的所有費用都是不應收費的。</span><span class="sxs-lookup"><span data-stu-id="fae72-125">The result is that all time recorded on the task is chargeable and all expenses recorded on the task are non-chargeable.</span></span>

<span data-ttu-id="fae72-126">工作的帳單類型可以在專案型報價明細的 **應收費工作** 索引標籤上，透過更新 **報價明細工作** 子格上的 **帳單類型** 欄位來進行設定。</span><span class="sxs-lookup"><span data-stu-id="fae72-126">A task's billing type can be configured on the **Chargeable Tasks** tab of a project-based quote line by updating the **Billing Type** field on the **Quote Line Tasks** subgrid.</span></span> <span data-ttu-id="fae72-127">或者，也可以在顯示與工作相關之報價明細的專案工作帳單設定子格的 **帳單類型** 欄位中更新專案工作的帳單類型。</span><span class="sxs-lookup"><span data-stu-id="fae72-127">Alternatively, you can update the billing type for a project task in the **Billing Type** field on the subgrid on the task billing setup of a project that shows the quote lines associated to a task.</span></span>
