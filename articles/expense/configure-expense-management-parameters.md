# <a name="configure-expense-management-parameters"></a>設定費用管理參數

_**適用於：** 資源/非庫存型案例適用的 Project Operations_

本主題描述控制費用管理中一般行為的參數。

## <a name="general"></a>一般

| 欄位                                                    | 描述 |
|----------------------------------------------------------|-------------|
| 里程的標準費率                                 | 輸入里程費用的報銷費率。 此費率會乘以費用輸入的里程，以計算償付差旅費用的金額。 |
| 驗證費用用途                                 | 開啟此選項，可將使用者限制在一組在其建立費用報表時於**費用報表用途**欄位中所設定的現有值。 |
| 個人費用支付者                                | 選取要支付任何分類為個人費用的信用卡交易金額的負責人。 |
| 在向上切入時顯示整份費用報表               | 選取此選項可在針對費用報表過帳時所產生的特定憑單來檢視原始單據的詳細資料時，顯示費用報表的所有費用。 |
| 強制差旅預先授權                 | 選取此選項，以要求員工必須先提交差旅申請並獲得核准，才能提交費用報表。 |
| 允許在過帳之前編輯分佈            | 選取是否可以在過帳費用報表之前，先編輯該報表的分佈。 |
| 評估費用管理原則                     | 選取評估費用的時機，以判斷是否已違反費用原則。 輸入並儲存費用項目時，或提交費用請求核准時，可以評估費用是否違規。 儲存費用明細時，將會評估所需收據的原則規則。 |
| 允許公司間的費用明細                         | 選取是否可以在費用報表上輸入其他法律實體的費用。 |
| 允許編輯信用卡費用的匯率 | 選取此選項以允許使用者編輯已匯入信用卡費用的匯率。 |
| 要顯示的多層級階層欄位                  | 選取當工作流程指派類型設定為階層，且**階層**選取項目設定為使用核准階層、費用多層核准時，顯示哪些核准者欄位。 工作流程使用多層核准階層時，所選取的欄位將會顯示在費用報表上，並且可加以編輯。 |
| 輸入員工信用卡編號                        | 選取是否可以輸入 15 字元或 16 字元號碼，並儲存在員工**信用卡**頁面的**卡片識別碼**欄位中 。 |
| 驗證差旅申請目的                      | 開啟此選項，可將使用者限制在一組在其建立差旅申請時於**費用報表用途**欄位中所設定的現有值。 |

## <a name="financial"></a>財務

| 欄位                                                                                    | 描述 |
|------------------------------------------------------------------------------------------|-------------|
| 總帳日記帳名稱                                                                | 選取已核准費用報表過帳到的總帳日記帳名稱。 |
| 啟用費用的稅金退還                                                        | 選取此選項可啟用合格費用的費用退稅。 如果已啟用美國銷售稅及使用課稅規則，則無法選取此選項。 |
| 立即過帳預付現金                                                           | 選取此選項可在完成支付和轉帳還時，將核准的預付現金預付過帳。 如果未選取此選項，則支付和轉帳程序會產生未過帳的一般帳目。 |
| 在過帳期間更正會計日期                                                   | 選取此選項，如果目前期間已暫停或已關帳，則可在過帳費用時更新會計日期。 會計日期將會設定為下一個開帳期間。 |
| 允許根據付款方式中指定的抵銷科目來群組交易       | 選取此選項可根據費用付款方式中指定的抵銷科目，摘要列出費用報表的費用交易。 |
| 包含稅金                                                                             | 選取此選項可根據預設，在費用金額中包含銷售稅。 |
| 關閉差旅申請時發還保留款                                      | 選取此選項可在關閉已核准的差旅申請時發還擔保資金。 |
| 允許在預算登記和專案預算的預算超支時提交差旅申請 | 選取此選項，讓員工即使在其超過預算登記中所設定的允許預算或專案允許的預算時，也能提交差旅申請。 |
| 允許在預算登記和專案預算的預算超支時提交費用報表     | 選取此選項，讓員工即使在其超過預算登記中所設定的允許預算或專案允許的預算時，也能提交費用報表。 |
| 只允許在預算登記的預算超支時核准費用報表                 | 選取此選項以允許核准者核准超過預算登記中所設定允許的預算的費用報表。 |

## <a name="per-diem"></a>按日

| 欄位                                 | 描述 |
|---------------------------------------|-------------|
| 獲取每日津貼至少所需的時數            | 輸入員工為符合獲得差旅相關津貼的資格，在一天中必須工作的最小基本時數。 這個值只會在每日津貼費率的**最小時數**欄位中用來做為預設值。 |
| 餐費百分比                          | 輸入當**計算餐費扣減依據**欄位設定為**每日餐飲類型**或**每日用餐次數**時，第一天和最後一天差旅相關費用上所用每日餐費津貼的預設百分比。 第一天和最後一天的工作日可能會比標準工作日短。 因此，這些工作日的差旅津貼金額可能會與標準金額不同。 如果百分比設定為 **0** (零)，則第一天和最後一天的扣減額會是 0.00。 |
| 住宿費百分比                         | 輸入第一天和最後一天差旅相關費用所用每日住宿津貼的預設百分比。 第一天和最後一天的工作日可能會比標準工作日短。 因此，這些工作日的差旅津貼金額可能會與標準金額不同。 如果百分比設定為 **0** (零)，則第一天和最後一天的扣減額會是 0.00。 |
| 其他百分比                         | 輸入第一天和最後一天差旅相關費用所用每日雜項開支津貼的預設百分比。 第一天和最後一天的工作日可能會比標準工作日短。 因此，這些工作日的差旅津貼金額可能會與標準金額不同。 如果百分比設定為 **0** (零)，則第一天和最後一天的扣減額會是 0.00。 |
| 早餐扣減百分比 | 輸入每日津貼就早餐所扣減的金額。 例如，如果員工獲得免費贈送的早餐，您可能會想要將每日差旅津貼的金額減少 10%。 |
| 午餐扣減百分比     | 輸入每日津貼就午餐所扣減的金額。 例如，如果員工獲得免費贈送的午餐，您可能會想要將每日差旅津貼的金額減少 15%。 |
| 晚餐扣減百分比    | 輸入每日津貼就晚餐所扣減的金額。 例如，如果員工獲得免費贈送的晚餐，您可能會想要將每日差旅津貼的金額減少 25%。 |
| 計算餐費扣減依據           | 選取根據每次出差還是每天的餐飲類型進行扣減，或根據每天參數的用餐次數進行扣減，以指定系統應如何計算每日餐費津貼扣減。 |
| 每日津貼捨入                     | 選取每日津貼費用所用的捨入類型。 如果您選取標準捨入，則金額為 0.50 的任何每加津貼費用都會四捨五入為 1.00，而金額小於 0.50 的每日津貼費用則向下捨入為 0.00。 |
| 每日津貼計算依據          | 選取每日津貼金額是根據日曆天還是 24 小時期間。 |

## <a name="fax-cover-pages"></a>傳真封面頁

| 欄位                          | 描述 |
|--------------------------------|-------------|
| 指示                   | 輸入員工在建立用於傳送與費用報表相關之收據的傳真封面頁時，必須遵循的指示。 若要根據使用者語言輸入顯示的語言特定文字，請選取**翻譯**。 |
| 使用者識別碼 (條碼資訊) | 選取此選項可將員工的唯一識別碼儲存在傳真封面頁上所使用的條碼中。 |
| 條碼                       | 選取傳真封面頁上使用的條碼。 費用管理支援條碼 39 和 128。 |

## <a name="anti-corruption"></a>反貪腐

| 欄位                                 | 描述 |
|---------------------------------------|-------------|
| 顯示反貪腐證明   | 選取此選項可在建立費用報表時顯示反貪腐文字。 可能會接著啟用需要您在費用報表上選取反貪腐證明的特定費用類別。 例如，與政府公務費用相關的禮品類別可能需要員工確認該費用符合與政府公務相關的公司原則。 |
| 給提交者的反貪腐訊息 | 輸入必須向建立費用報表的員工顯示的文字。 若要根據使用者語言輸入顯示的語言特定文字，請選取**翻譯**。 |
| 給核准者的反貪腐訊息  | 輸入必須在建立費用報表時向核准者顯示的文字。 若要根據使用者語言輸入顯示的語言特定文字，請選取**翻譯**。 |
