# <a name="recording-actuals"></a>記錄實際值 

[!INCLUDE[cc-applies-to-psa-app-3.x](../includes/cc-applies-to-psa-app-3x.md)]

系統會記錄所有在專案期間發生的財務交易。 這些交易記錄會以 **實際值** 來記錄。 下表顯示根據專案是時間及材料專案、固定價格專案還是內部專案，或是否在售前階段，所建立的不同類型實際值。
