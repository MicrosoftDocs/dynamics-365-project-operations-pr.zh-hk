# <a name="configure-transaction-categories-project-service"></a>設定交易類別 (Project Service)

[!INCLUDE[cc-applies-to-psa-app-1x-2x](../includes/cc-applies-to-psa-app-1x-2x.md)]

設定交易類別以定義費用類別的超集。  
  
1.  移至 **Project Service > 交易類別**。  
  
2.  按一下 **新增**。  
  
3.  在 **一般** 區域中，視需要填入欄位。  
  
4.  按一下畫面右下角的 **儲存**。
