# <a name="invoice-schedules-on-project-based-quote-lines"></a><span data-ttu-id="59794-103">專案型報價明細的發票排程</span><span class="sxs-lookup"><span data-stu-id="59794-103">Invoice schedules on project-based quote lines</span></span>

<span data-ttu-id="59794-104">_**適用於：** 資源/非庫存型案例適用的 Project Operations、精簡部署 - 交易至開立預估發票_</span><span class="sxs-lookup"><span data-stu-id="59794-104">_**Applies To:** Project Operations for resource/non-stocked based scenarios, Lite deployment - deal to proforma invoicing_</span></span>

## <a name="create-a-time-and-material-invoice-schedule-for-a-project-based-quote-line"></a><span data-ttu-id="59794-111">建立專案型報價明細的時間和材料發票排程</span><span class="sxs-lookup"><span data-stu-id="59794-111">Create a Time and material invoice schedule for a project-based quote line</span></span>

<span data-ttu-id="59794-112">專案型報價明細的帳務方式為 [時間和材料] 時，系統會產生基於日期的發票排程。</span><span class="sxs-lookup"><span data-stu-id="59794-112">When the billing method for a project-based quote line is Time and material, the system generates a date-based invoice schedule.</span></span> <span data-ttu-id="59794-113">若要讓系統自動產生基於日期的發票排程，請完成下列步驟。</span><span class="sxs-lookup"><span data-stu-id="59794-113">To automatically generate a date-based invoice schedule, complete the following steps.</span></span>

2. <span data-ttu-id="59794-115">在**報價**頁面上，開啟專案報價，並在**摘要**索引標籤中設定要求的交付日期。</span><span class="sxs-lookup"><span data-stu-id="59794-115">On the **Quotes** page, open the Project quote and on the **Summary** tab, set a requested delivery date.</span></span>
