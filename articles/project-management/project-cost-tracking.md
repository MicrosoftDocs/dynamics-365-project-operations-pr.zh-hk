# <a name="labor-cost-tracking-on-projects"></a><span data-ttu-id="d318c-103">專案的人力成本追蹤</span><span class="sxs-lookup"><span data-stu-id="d318c-103">Labor cost tracking on projects</span></span>

<span data-ttu-id="d318c-104">_**適用於：** 資源/非庫存型案例適用的 Project Operations、精簡部署 - 交易至開立預估發票_</span><span class="sxs-lookup"><span data-stu-id="d318c-104">_**Applies To:** Project Operations for resource/non-stocked based scenarios, Lite deployment - deal to proforma invoicing_</span></span>

<span data-ttu-id="d318c-105">Dynamics 365 Project Operations 會以專案計劃中所需的最低資料粒度來追蹤人力估計值及支出。</span><span class="sxs-lookup"><span data-stu-id="d318c-105">Dynamics 365 Project Operations tracks labor estimates and spend at the lowest required granularity on the project plan.</span></span> <span data-ttu-id="d318c-106">人力成本的財務估計是根據指派給專案計劃中每個分葉節點工作的計劃投入量和一般或具名資源而定。</span><span class="sxs-lookup"><span data-stu-id="d318c-106">The financial estimate of labor cost is based on the planned effort and the generic or named resources that are assigned to each leaf-node task in the project plan.</span></span> <span data-ttu-id="d318c-107">當專案開始且人員開始報告各種不同專案工作的時間時，就會彙總人力的實際支出，而這將開始計算推估。</span><span class="sxs-lookup"><span data-stu-id="d318c-107">When the project begins and people start to report time on various project tasks, the actual spend on labor is summarized which starts a calculation of the projections.</span></span>

## <a name="labor-cost-tracking-view"></a><span data-ttu-id="d318c-108">人力成本追蹤檢視表</span><span class="sxs-lookup"><span data-stu-id="d318c-108">Labor Cost Tracking view</span></span>

<span data-ttu-id="d318c-109">在 **專案** 頁面的 **追蹤** 索引標籤中，您可以選取 **追蹤** > **成本** 來開啟 **成本追蹤** 檢視表，並在專案計劃中查看每項工作的人力支出進度。</span><span class="sxs-lookup"><span data-stu-id="d318c-109">On the **Projects** page, on the **Tracking** tab, you can select **Tracking** > **Cost** to open the **Cost Tracking** view and see the progress of labor spend on each task in the project plan.</span></span> <span data-ttu-id="d318c-110">此檢視表也會將工作所花費的實際人力成本與工作的計劃人力成本進行比較。</span><span class="sxs-lookup"><span data-stu-id="d318c-110">This view also compares the actual labor cost spent on a task to the task's planned labor cost.</span></span> <span data-ttu-id="d318c-111">Project Operations 使用下列公式計算人力成本計量：</span><span class="sxs-lookup"><span data-stu-id="d318c-111">Project Operations uses the following formulas to calculate labor cost metrics:</span></span>

- <span data-ttu-id="d318c-112">**計劃成本**：每個分葉節點工作上所有資源指派的估計成本</span><span class="sxs-lookup"><span data-stu-id="d318c-112">**Planned cost**: Estimated cost of all resource assignments on each leaf node task</span></span>
- <span data-ttu-id="d318c-113">**實際成本**：工作所記錄時間的所有成本實際值的加總</span><span class="sxs-lookup"><span data-stu-id="d318c-113">**Actual Cost**: Sum of all cost actuals for time recorded on the task</span></span>
- <span data-ttu-id="d318c-114">**成本耗用百分比**：實際成本 ÷ 成本完工估計值</span><span class="sxs-lookup"><span data-stu-id="d318c-114">**Cost consumption percentage**: Actual cost ÷ Cost estimate at complete</span></span>
- <span data-ttu-id="d318c-115">**剩餘成本**：成本完工估計值 – 實際成本</span><span class="sxs-lookup"><span data-stu-id="d318c-115">**Remaining Cost**: Cost estimate at complete  – Actual cost</span></span>
- <span data-ttu-id="d318c-116">**完工成本**：剩餘成本 + 實際成本</span><span class="sxs-lookup"><span data-stu-id="d318c-116">**Cost at Complete**: Remaining cost + Actual cost</span></span>
- <span data-ttu-id="d318c-117">**成本差異**：計劃成本 – 成本完工估計值</span><span class="sxs-lookup"><span data-stu-id="d318c-117">**Cost variance**: Planned cost – Cost estimate at complete</span></span>

<span data-ttu-id="d318c-118">每項工作都會顯示對工作的成本差異推估。</span><span class="sxs-lookup"><span data-stu-id="d318c-118">Each task shows a projection of the cost variance on the task.</span></span> <span data-ttu-id="d318c-119">如果成本完工估計值大於計劃成本，就會將工作推估為超出預算。</span><span class="sxs-lookup"><span data-stu-id="d318c-119">If the cost estimate at complete is more than the planned cost, the task is projected to go over budget.</span></span> <span data-ttu-id="d318c-120">如果成本完工估計值小於計劃成本，則將工作推估為在預算下完成。</span><span class="sxs-lookup"><span data-stu-id="d318c-120">If the cost estimate at complete is less than the planned cost, the task is projected to finish under budget.</span></span>

>[!NOTE]
> <span data-ttu-id="d318c-121">Project Operations 只會在 **追蹤** 索引標籤的 **專案** 頁面上顯示人力成本。雖然可以估計並追蹤材料和費用的耗用，但是這些成本不會包含 **追蹤** 索引標籤所顯示的成本中。此索引標籤的設計只適用於透過投入量重推估來重新推估人力成本。</span><span class="sxs-lookup"><span data-stu-id="d318c-121">Project Operations only shows labor costs on the **Project** page on the **Tracking** tab. This tab is designed to work only for reprojecting labor costs by reprojecting effort.</span></span>
<span data-ttu-id="d318c-122">顯示的所有成本金額都會從專案用於決定成本費率的成本價貨幣轉換為專案的成本貨幣。</span><span class="sxs-lookup"><span data-stu-id="d318c-122">All cost amounts shown are converted to the project's cost currency from the project's cost price currency used to determine the cost rate.</span></span> <span data-ttu-id="d318c-123">專案的成本貨幣即是專案中合約單位的貨幣。</span><span class="sxs-lookup"><span data-stu-id="d318c-123">The cost currency of the project is the currency of the contracting unit on the project.</span></span> <span data-ttu-id="d318c-124">**專案** 頁面的 **估計值** 索引標籤上所顯示的時間估計成本值可能不會加總到 **追蹤** 索引標籤上的計劃成本。產生這種不一致情况的原因是，在 **估計值** 網格上彙總估計成本方式與在 **追蹤** 網格中計算計劃成本方式中的差異。</span><span class="sxs-lookup"><span data-stu-id="d318c-124">The estimated cost values for time that are shown on the **Estimates** tab on the **Project** page might not add up to the planned cost on the **Tracking** tab. >
> - <span data-ttu-id="d318c-125">**估計值索引標籤** 使用與價目表中成本費率相同的貨幣來計算估計成本。</span><span class="sxs-lookup"><span data-stu-id="d318c-125">The **Estimates tab** calculates the estimated cost by using the same currency of the cost rate in the price list.</span></span> <span data-ttu-id="d318c-126">然後，以價目表貨幣計價的估計成本會轉換為以專案成本貨幣計價的估計成本。</span><span class="sxs-lookup"><span data-stu-id="d318c-126">Then, the estimated cost in the price list currency is converted to the estimated cost in the project's cost currency.</span></span> <span data-ttu-id="d318c-127">以專案貨幣計價的估計成本會顯示為捨入至 2 個小數位數。</span><span class="sxs-lookup"><span data-stu-id="d318c-127">The estimated cost in the project currency is shown rounded to 2 decimal places.</span></span> <span data-ttu-id="d318c-128">在此轉換過程中，任何時點都不會套用貨幣有效位數。</span><span class="sxs-lookup"><span data-stu-id="d318c-128">At no point during this conversion is currency precision applied.</span></span> 
> - <span data-ttu-id="d318c-129">在 **追蹤** 索引標籤上，計劃成本計算依循略微不同的計算順序，包括分兩個階段套用貨幣有效位數：</span><span class="sxs-lookup"><span data-stu-id="d318c-129">On the **Tracking** tab, the planned cost calculation follows a slightly different calculation order that involves currency precision being applied in two stages:</span></span> 
   ><ol>
   ><li><span data-ttu-id="d318c-130">以價目表貨幣計價中的估計成本金額轉換為基準貨幣 (第 1 次轉換)。</span><span class="sxs-lookup"><span data-stu-id="d318c-130">The estimated cost amount in the price list currency is converted to the base currency (conversion 1).</span></span></li>
   ><li><span data-ttu-id="d318c-131">以基準貨幣計價的估計成本金額轉換為專案成本貨幣 (第 2 次轉換)。</span><span class="sxs-lookup"><span data-stu-id="d318c-131">The estimated cost amount in the base currency is converted to the project cost currency (conversion 2).</span></span> </li>
