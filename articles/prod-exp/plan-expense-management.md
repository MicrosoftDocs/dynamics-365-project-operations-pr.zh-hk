# <a name="configure-expense-management"></a>設定費用管理

本主題說明設定費用管理之前，您必須在規劃過程中進行的考量和決策。 在費用管理中，您可以儲存付款方式、差旅申請、費用報表、原則等相關資訊。

因為在規劃費用管理設定時所做的許多決策是根據組織的階層和財務結構而定，您必須參考這些區域的計劃文件。

## <a name="intercompany-expenses"></a>公司間費用

啟用公司間費用時，您可以允許法律實體和員工代表其他法律實體承擔費用，並收取該法律實體支付組織內雇用的款項。 例如，法律實體 A 中的員工為法律實體 B 完成專案，而此專案產生差旅相關的費用。 如果啟用公司間費用，員工就可以提出會將費用過帳至法律實體 B 的費用報表，而且該費用必須由法律實體 A 支付。如果您的組織沒有多個法律實體，則不需要啟用公司間費用。

**決策：** 是否要啟用公司間費用？

## <a name="financial-management"></a>財務管理

費用管理與組織的財務管理緊密整合在一起。 費用管理的許多設定都會依據您所做關於組織財務的決策而定。 下列各節說明各個需要根據組織財務決策和領導團隊指導進行規劃和決策的領域。

### <a name="per-diems"></a>每日津貼

您必須定義組織提供給員工的每日津貼。 因為每日津貼通常用來抵補餐費、住宿費和其他雜項費用等開支，您可以為組織所提供的差旅津貼補助建立規則。 每日津貼費率可以根據一年時節、出差地點或兩者來定。 定義每日津貼規則時，您可以指定如果工作者獲得免費贈送的餐飲或服務時，每日津貼費率會扣留的百分比。 您也可以每日津貼費率分級來設定每日津貼費率可套用至工作者差旅的最小及最大時數。

**決策：**

- 第一天和最後一天的預設每日津貼規則：

    - 員工可報銷不足一天的時數，而仍將收到足額每日津貼的最少時數為何？
    - 是否會扣減已提供的第一天與最後一天餐費金額？ 如果有扣減，扣減百分比為何？
    - 是否會扣減已提供的第一天與最後一天住宿金額？ 如果有扣減，扣減百分比為何？
    - 是否會扣減已提供的第一天與最後一天費用開支金額？ 如果有扣減，扣減百分比為何？

- 預設每日津貼規則：

    - 如果說出差員工享有免費供餐，是否會從每餐的每日津貼補助中扣減一定的百分比？ 如果有扣減，每餐的扣減百分比為何？
    - 餐費扣減是按每日、每趟差旅還是每天用餐次數來計算？
    - 每日津貼金額是依照一般的四捨五入方式還是向上捨入？
    - 每日津貼是按照 24小時週期還是日曆日進行計算？

- 根據地點而定的每日津貼規則：

    - 每日津貼費率是否會依據地點而有所不同？ 包括哪些地點？
    - 如果每日津貼費率依地點而不同，則對於每個地點，下列費用類型提供的百分比金額為何：

        - 餐費
        - 飯店
        - 其他費用

### <a name="expense-management-journals-and-accounts"></a>費用管理日記帳和科目

費用管理要求您必須使用多個日記帳和科目。 例如，您必須決定是否要將相同的科目用於預付現金和信用卡紛爭。

**決策：**

- 已核准的費用報表會過帳到哪些總帳日記帳？
- 預付現金要使用哪一個科目？
- 預付現金是否應立即過帳？

### <a name="payment-methods"></a>付款方式

允許員工代表公司開支費用時，您必須定義員工可以使用的付款方式。 例如，您可能會允許員工使用現金或公司信用卡來支付。 您可能也會允許員工先用個人信用卡支付，然後您再補償員工。 您必須對每個您所允許的付款方式做出下列決策。

**決策：**

- 允許使用哪些付款方式？
- 誰負責付款方式費用？
- 是否有抵銷科目類型？ 如果有抵銷科目類型，是什麼類型？
- 如果有抵銷科目，是什麼科目？
- 如果付款方式為信用卡，付款方式只能用於匯入交易嗎？

### <a name="expense-categories-and-shared-categories"></a>費用類別和共用類別

員工建立費用報表時，他們記錄的每項費用都必須與費用類別有關聯。 費用類別是從可在您組織中所有法律實體之間共用的共用類別衍生而來。 視組織的定義方式而定，也可以在 [專案管理與會計] 中共用這些類別。 請根據組織的定義和實作團隊的指引，判斷費用管理中使用的類別是否僅限在 [費用管理] 中使用，或是否應在 [專案管理與會計] 與 [費用管理] 之間共用。 請注意，這些類別可以在 [專案] 與 [費用] 之間或 [專案] 與 [生產] 之間共用，但無法在 [費用] 與 [生產] 之間共用。 您必須對每個費用類別做出下列決策。

**決策：**

- 什麼是費用類別？ 範例包括航班、住宿或里程等類別。
- 費用類別也可以用於 [專案管理與會計] 嗎？
- 什麼是費用類型？
- 什麼是費用類別的預設付款方式？
- 費用類別中的費用是否必須逐條列舉？
- 什麼是費用類別的主要預設科目？
- 什麼是費用類別的預設項目銷售課稅群組？
- 費用類別是否允許其他付款方式？ 如果允許其他付款方式，是哪些方式？
- 此費用類別中是否有子類別？ 如果有子類別，您還必須做出下列決定：

    - 是否有任何子類別不在稅金退還之列？
    - 什麼是子類別的項目銷售課稅群組？

如果也會在 [專案管理與會計] 中使用費用類別，請回答剩下的問題。 否則，請移至下一個區段。

- 下列費用會使用哪個成本科目？

    - 成本
    - 薪資分攤
    - WIP 成本值
    - 成本項目
    - WIP 成本值項目
    - 應計損失
    - WIP 應計損失

- 下列各項會使用哪個收入科目？

    - 已開發票營收
    - 應計收入 - 銷售值
    - WIP - 銷售值
    - 應計收入 - 生產
    - WIP - 生產
    - 應計收入 - 利潤
    - WIP - 利潤
    - 應計收入 - 訂閱
    - WIP - 訂閱

### <a name="taxes"></a>稅金

對於費用相關稅金，您必須判斷費用報表中所包含或啟用的項目。

**決策：**

- 銷售稅是否包含在費用金額中？
- 是否應在費用上啟用稅金退還？

    > [!NOTE]
    > 規劃總帳當時，如果您已決定套用美國銷售稅並使用課稅規則，則無法在費用上啟用稅金退還。 (若要套用美國銷售稅並使用課稅規則，請將 **套用銷售稅課稅規則** 選項設定為 **是**)。

## <a name="policies"></a>原則

您可以建立費用報表原則，協助組織在員工代表其開支費用時節省時間和金錢。 原則有助於保證員工不超出預算、提供所有必要資訊，並且只在需要時花費資金。 您必須對每個費用報表原則以及每個您所建立的費用報表核准原則做出下列決定。

**決策：**

- 原則的名稱是什麼？
- 費用原則的目的是什麼？
- 如果先前已決定要啟用公司間費用，您組織中的哪些公司將會套用此原則？
- 原則何時生效？
- 原則何時到期？
- 什麼是原則規則？
- 原則規則的結果是什麼？
