# <a name="configure-accounting-for-billable-projects"></a><span data-ttu-id="87573-103">設定計費專案會計</span><span class="sxs-lookup"><span data-stu-id="87573-103">Configure accounting for billable projects</span></span>

<span data-ttu-id="87573-104">_**適用於：** 資源/非庫存型案例適用的 Project Operations、精簡部署 - 交易至開立預估發票_</span><span class="sxs-lookup"><span data-stu-id="87573-104">_**Applies To:** Project Operations for resource/non-stocked based scenarios, Lite deployment - deal to proforma invoicing_</span></span>

<span data-ttu-id="87573-105">Dynamics 365 Project Operations 針對包含時間和材料以及固定價格交易的計費專案，支援各種不同的會計選項。</span><span class="sxs-lookup"><span data-stu-id="87573-105">Dynamics 365 Project Operations supports various accounting options for billable projects that include time and material and fixed price transactions.</span></span>

- <span data-ttu-id="87573-106">**時間和材料交易**：根據專案上時數、費用、項目或服務費的耗用，這些交易會依照工作進度開立發票。</span><span class="sxs-lookup"><span data-stu-id="87573-106">**Time and material transactions**: These transactions are invoiced as the work progresses based on the consumption of hours, expenses, items, or fees on the project.</span></span> <span data-ttu-id="87573-107">這些交易成本可能會與每筆交易的營收等值，而且也會隨工作進展，開立專案的發票。</span><span class="sxs-lookup"><span data-stu-id="87573-107">These transaction costs can be matched with the revenue on each transaction and the project is invoiced as work progresses.</span></span> <span data-ttu-id="87573-108">專案營收也可以在交易發生之時列入應計項目。</span><span class="sxs-lookup"><span data-stu-id="87573-108">Project revenue can be also accrued at the time when the transaction occurs.</span></span> <span data-ttu-id="87573-109">開立發票時，會認列營收並在適用時沖回應計收入。</span><span class="sxs-lookup"><span data-stu-id="87573-109">During invoicing, revenue is recognized and if applicable, accrued revenue is reversed.</span></span>
- <span data-ttu-id="87573-110">**固定價格交易**：這些交易是根據以專案合約為基準的帳單排程來開立發票。</span><span class="sxs-lookup"><span data-stu-id="87573-110">**Fixed-price transactions**: These transactions are invoiced according to a billing schedule that is based on the project contract.</span></span> <span data-ttu-id="87573-111">固定價格交易的營收可以依據**全部完工**或**完工百分比**方法，在開立發票時認列，或定期計算並過帳。</span><span class="sxs-lookup"><span data-stu-id="87573-111">Revenue for fixed price transactions can be recognized at invoicing or calculated and posted periodically, according to the **Completed contract** or **Completed percentage** methods.</span></span>

<span data-ttu-id="87573-112">當專案與一個或多個合約服務內容相關聯時，將其視為計費專案。</span><span class="sxs-lookup"><span data-stu-id="87573-112">A project is considered billable when it is associated with one or more contract lines.</span></span> <span data-ttu-id="87573-113">專案合約服務內容會為其本身定義允許哪一種帳務方式和交易類型。</span><span class="sxs-lookup"><span data-stu-id="87573-113">A project contract line defines for itself which billing method and transaction types are allowed.</span></span>

<span data-ttu-id="87573-114">例如，Fabrikam Robotics 已爭取到與 Adatum Corporation 簽訂設備最佳化的專案合約。</span><span class="sxs-lookup"><span data-stu-id="87573-114">As an example, Fabrikam Robotics has won a project contract with Adatum corporation for equipment optimization.</span></span> <span data-ttu-id="87573-115">Adatum 將支付 $10.000 美元的固定金額，以承擔產生的專案費用。</span><span class="sxs-lookup"><span data-stu-id="87573-115">Adatum will pay a fixed amount of $10.000 USD to cover incurred project expenses.</span></span> <span data-ttu-id="87573-116">這是固定價格帳務方式。</span><span class="sxs-lookup"><span data-stu-id="87573-116">This is a fixed price billing method.</span></span> <span data-ttu-id="87573-117">專案時間和費用將按每次使用計費。</span><span class="sxs-lookup"><span data-stu-id="87573-117">Project time and fees will be billed per use.</span></span> <span data-ttu-id="87573-118">這是時間和材料帳務方式。</span><span class="sxs-lookup"><span data-stu-id="87573-118">This is a time and material billing method.</span></span> <span data-ttu-id="87573-119">所有的工作都會在名為「Adatum 設備最佳化」的單一專案下進行追蹤。</span><span class="sxs-lookup"><span data-stu-id="87573-119">All of the work will be tracked under a single project named, Adatum equipment optimization.</span></span>

<span data-ttu-id="87573-120">專案團隊成員提交在 Adatum 設備最佳化專案上工作 8 小時的時數。</span><span class="sxs-lookup"><span data-stu-id="87573-120">A project team member submits eight hours of work on the Adatum equipment optimization project.</span></span> <span data-ttu-id="87573-121">專案經理核准此時間項目時，系統會使用時間和材料帳務方式來建立實際值交易、發票和科目。</span><span class="sxs-lookup"><span data-stu-id="87573-121">When the project manager approves this time, the system uses the time and material billing method to create actuals transactions, an invoice, and an account.</span></span> <span data-ttu-id="87573-122">此交易不會納入固定價格營收估計值計算中。</span><span class="sxs-lookup"><span data-stu-id="87573-122">This transaction will not be included in the fixed price revenue estimate calculation.</span></span>

<span data-ttu-id="87573-123">另一個專案團隊成員針對 Adatum 設備最佳化專案提交 $2000.00 美元的差旅費用。</span><span class="sxs-lookup"><span data-stu-id="87573-123">Another project team member submits a travel expense for $2000.00 USD against the Adatum equipment optimization project.</span></span> <span data-ttu-id="87573-124">專案經理核准此提交項目時，系統會使用固定價格帳務方式來建立此專案費用的實際值交易和科目。</span><span class="sxs-lookup"><span data-stu-id="87573-124">When the project manager approves this submission, the system uses a fixed price billing method to create actuals transactions and an account for this project expense.</span></span> <span data-ttu-id="87573-125">將不會根據此交易開立發票給客戶。</span><span class="sxs-lookup"><span data-stu-id="87573-125">The customer will not be invoiced based on this transaction.</span></span> <span data-ttu-id="87573-126">反而會使用另外設定的帳單里程碑來為他們開立發票。</span><span class="sxs-lookup"><span data-stu-id="87573-126">Instead, they will be invoiced by using separately configured billing milestones.</span></span> <span data-ttu-id="87573-127">此費用交易以及費用估計值不會納入固定價格營收估計值計算中。</span><span class="sxs-lookup"><span data-stu-id="87573-127">This expense transaction, along with expense estimates, will be included in the fixed price revenue estimate calculation.</span></span>

<span data-ttu-id="87573-128">專案交易的會計原則是在專案成本與營收設定檔中定義。</span><span class="sxs-lookup"><span data-stu-id="87573-128">Accounting principles for project transactions are defined in project cost and revenue profiles.</span></span> <span data-ttu-id="87573-129">對於每筆專案交易，系統都會使用已設定的專案成本與營收設定檔規則，判斷適當的專案成本與營收設定檔。</span><span class="sxs-lookup"><span data-stu-id="87573-129">For every project transaction, the system determines the appropriate project cost and revenue profile by using the project cost and revenue profile rules that have been configured.</span></span>
